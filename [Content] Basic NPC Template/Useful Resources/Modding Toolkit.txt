RECOMMENDED EDITORS:
        Visual Studio Code: https://code.visualstudio.com/
            Pros: Easy extension support
            Cons: Bloated with Microsoft extras
    OR
        Visual Studio Codium: https://vscodium.com/
            Pros: Open source, lighter
            Cons: Less straightforward extension support

    Important Additions: 
        Stardew Syntax Extension: https://marketplace.visualstudio.com/items?itemName=linkoid.stardew-syntax
        JSON with Comments: (Setting found in bottom right corner, "Select Language Mode" on hover)

    SUGGESTED NOT TO USE:
        Notepad++ 
            Harder to use
            Less optimized for SDV
            No error checking out of the box
        Visual Studio
            Commonly accidentally confused for Visual Studio Code
            Very heavy on PC memory
            Harder to navigate
            Oriented more for C# modders

USEFUL CONSOLE COMMANDS:
    Full List: https://stardewvalleywiki.com/Modding:Console_commands
        reload_i18n                            Force reloads your i18n changes                  
        patch reload MODID                     Force reloads your entire mod
        patch reload ModID FilePath            Force reloads a specific file in your mod
        patch summary MODID full               Shows a list of all your tokens and patches and if they're being applied without shortened tokens
        patch summary full                     Shows a list of ALL tokens and patches added by all installed mods without shortened tokens
        patch summary full asset InternalFile  Shows patches affecting a specific asset
        patch export Internalfile              Exports the data currently active in the game. Good to see if things are loading.
        debug ebi EventID                      Force plays an event regardless of conditions
        debug ee                               Force ends and event or festival
        debug nd                               Forces the day to end and update the world state
        debug festival seasonday               Forces a specific festival to load regardless of season
        debug wtc NPCID                        Warps player to specified NPC
        debug wctm NPCID                       Warps specified NPC to player
        debug killNPC NPCID                    Removes NPC for the day, useful if they decided to duplicate
        debug warp location                    Warps player to specified location
        debug clearmail                        Wipes all current active flags  
        debug seenmail MailID add|remove       Marks a specific flag as seen without needing to naturally activate it
        debug gq GSQhere                       Runs a GSQ and displays the current state of the condition on the save (https://stardewvalleywiki.com/Modding:Game_state_queries)
        debug loaddialogue NPCID Key           Force loads a specific NPC dialogue
        debug where NPCID                      Locates an NPC and displays their location and x, y coordinates
        debug time time                        Sets to specified time     
        debug season season                    Sets to specified season
        debug day #                            Sets specified day of month
        debug clear                            Clears player inventory
        debug action TriggerAction             Force runs a specific Trigger Action (https://stardewvalleywiki.com/Modding:Trigger_actions)

        Note: Commands that change the time/season/day may cause instabilities. Consider using CJB Cheats.
